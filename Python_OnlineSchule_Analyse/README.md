# Online School CRM Data Analysis

## Dataset

The dataset used in this project originates from the CRM system of an online programming school.  
Due to data privacy restrictions, the original data files are not included in this repository.

The analysis is based on four datasets:

- **leads.xlsx** – information about incoming leads, including source, stage, and quality
- **deals.xlsx** – data about deals created in the CRM, including deal status and value
- **payments.xlsx** – payment information related to closed deals
- **marketing_campaigns.xlsx** – information about marketing campaigns and lead sources

---

## Project Overview

This project analyzes CRM data from an online programming school.  
The goal of the analysis is to clean and explore lead and deal data in order to identify factors that influence sales performance and conversion rates.

The project demonstrates a full data analytics workflow including data cleaning, exploratory data analysis (EDA), and business analysis using Python.

---

## Business Context

An online programming school collects data in its CRM system about leads, deals, marketing sources, and sales activity.

As a data analyst, the task is to analyze this data to understand:

- how leads convert into deals
- which marketing channels generate the most effective leads
- how long deals take to close
- what factors influence successful sales

The results of the analysis can help improve marketing strategy and sales performance.

---

## Project Objectives

The main objectives of the project were:

- Clean and preprocess CRM data
- Analyze deal creation and closing trends
- Evaluate lead sources and campaign effectiveness
- Analyze sales funnel performance
- Identify key drivers of successful deals
