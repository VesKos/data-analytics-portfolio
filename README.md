# 🐍 Python Projects

### 📊 Online School CRM Analysis
Analysis of CRM data from an online programming school.

Key tasks:

- data cleaning and preprocessing  
- exploratory data analysis  
- lead source performance analysis  
- deal conversion analysis  
- sales funnel insights  

Tools: **Python, Pandas, Matplotlib, Seaborn**

---

### 📈 Revenue Forecast Using ARIMA

Time series forecasting project analyzing e-commerce revenue dynamics.

Key steps:

- time series exploration  
- stationarity testing (ADF test)  
- first differencing transformation  
- ARIMA model identification and selection  
- short-term revenue forecasting  

Tools: **Python, Pandas, Statsmodels, Matplotlib**

---

### 🎬 Film Search Automation

A Python automation project for searching and processing film-related data.

The project demonstrates:

- data processing
- automation scripts
- API interaction
- Python scripting for data workflows

⚠️ **Security Note**

The file `settings_bot.py` containing sensitive information (such as API tokens) is not included in the repository.

To run the project, create the file manually and add the required variables:
TOKEN = "your_token"
